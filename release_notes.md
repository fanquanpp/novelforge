# NovelForge v1.0.0

离线小说创作工作站 - 首个正式版本

## 功能特性

### 项目管理
- 四种创作题材模板：西幻史诗、标准长篇、散文随笔、舞台剧本
- 不同类型生成不同子目录结构与预设文件
- 支持本地项目导入与扫描
- FANDEX 三色系暗黑主题（蓝/绿/橙）

### 工作台
- 三栏布局：左侧导航 + 中间文件列表 + 右侧内容区
- 七大分类：正文、角色、世界观、名词、时间线、大纲、素材
- 角色/世界观/名词卡片式管理（增删改查）
- 时间线管理（竖线展示、分支筛选、事件 CRUD）

### 富文本编辑器
- 基于 TipTap 的富文本编辑
- 工具栏：加粗/斜体/标题/列表/引用/撤销/重做
- 自动保存（30 秒 + Ctrl+S）
- 实时字数统计（中文 + 英文单词）
- Markdown 与 HTML 互转

### 特色自动化功能
- **剧本类型**：空行按 Tab 键呼出角色名选择浮层，支持自定义输入
- **散文类型**：自动首行双字缩进（全角空格）
- **诗歌排版**：Ctrl+Shift+P 应用居中、行间距加大样式
- **歌词排版**：Ctrl+Shift+L 应用左侧装饰、节间分隔样式
- **TXT 导出**：一键导出纯文本文件

### 主题切换
- 亮色/暗色主题切换
- 状态持久化到 localStorage
- FANDEX 设计令牌完整支持

## 技术栈
- Tauri 2.0 + React 18 + TypeScript 5
- Tailwind CSS 3.4 + TipTap + Zustand + Framer Motion
- Rust 1.95.0 后端

## 安装包
- `NovelForge_1.0.0_x64-setup.exe`：NSIS 安装包（2.8 MB）
- `NovelForge_1.0.0_x64_en-US.msi`：MSI 安装包（4.13 MB）

## 系统要求
- Windows 10/11 x64
- WebView2 Runtime（Windows 11 已内置）
